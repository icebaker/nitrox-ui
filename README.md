# Nitrox UI

The graphical user interface for [Nitrox](https://github.com/icebaker/nitrox).

Check the [Nitrox documentation](https://icebaker.github.io/nitrox) for more information.

## Development

```bash
npm install

cp .env.example .env

./init.sh
```

## Build

```bash
npm run build
npm run preview
```
